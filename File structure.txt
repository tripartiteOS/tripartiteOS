
In tripartiteOS, the file structure should be similar to this:

C:
{
	AUTOEXEC.BAT
	CONFIG.SYS
	PATHCFG.BAT
	IO.SYS
	MSDOS.SYS
	DOS
	{
		[MS-DOS SYSTEM FILES]
	}
	TRPRTOS
	{
		SYSTEM.INI
		TRPRTOS.INI
		SYSTEM
		{
			TOS.COM
			KERNEL.EXE
			CWSDPR0.EXE
			USERINT.EXE
			DRIVERS
			{
				COM2EXE.EXE
				CTM-BR.EXE
				CTM-DE.EXE
				CTMDEBUG.EXE
				CTM-EN.EXE
				CTM-ES.EXE
				CTM-FR.EXE
				CTM-HU.EXE
				CTM-IT.EXE
				CTM-NL.EXE
				CTMOUSE.EXE
				CTM-PL.EXE
				CTM-PT.EXE
				CTM-SK.EXE
				HDPMI32i.EXE
				HIMEMSX.EXE
				JEMM386.EXE
				JLOAD.EXE
				QPIEMU.DLL
				SBEMU.EXE

				SRDUMMY.SYS
				SRDXMS.SYS
				SRDXMSS.SYS
                UNISOUND.COM
				XCDROM.SYS
				XMSSIZE.EXE
			}
			DRVINIT
			{
				AUDMDRN.BAT
				AUDOLD.BAT
				CDROM.BAT

				MOUSEWHL.BAT

			{
				REQSYSRB.COM
			}
		}
	}
}

Description:

C:
Your system drive

	AUTOEXEC.BAT
	Programs that run on startup
	
	CONFIG.SYS
	Drivers that start on startup
	
	IO.SYS
	MS-DOS system file
	
	MSDOS.SYS
	MS-DOS system file
	
	DOS/
	MS-DOS system files
	
	TRPRTOS/
	Main directory of tripartiteOS
	
		SYSTEM.INI
		tripartiteOS configuration file, contains different system-related parameters
		
		TRPRTOS.INI
		tripartiteOS configuration file, contains different usermode application and system component parameters

		SYSTEM/
		tripartiteOS's system file directory
	
			TOS.COM
			tripartiteOS real-mode initialization program
		
			KERNEL.EXE
			tripartiteOS's protected mode kernel
		
			CWSDPR0.EXE
			Ring 0 DPMI
		
			USERINT.EXE
			tripartiteOS's User Interface (NOT the shell)
		
			DRIVERS/
			tripartiteOS's drivers

				COM2EXE.EXE
				CTM-BR.EXE
				CTM-DE.EXE
				CTMDEBUG.EXE
				CTM-EN.EXE
				CTM-ES.EXE
				CTM-FR.EXE
				CTM-HU.EXE
				CTM-IT.EXE
				CTM-NL.EXE
				CTMOUSE.EXE
				CTM-PL.EXE
				CTM-PT.EXE
				CTM-SK.EXE
				CuteMouse's driver files

				HDPMI32i.EXE
				JEMM386.EXE
				JLOAD.EXE
				QPIEMU.DLL
				SBEMU.EXE
				SBEMU's driver files

				HIMEMSX.EXE
				Super Extended Memory Manager

				SRDEMS.SYS
				SRDEMS3.SYS
				SRDEMSS.SYS
				SRDISK.EXE
				SRDUMMY.SYS
				SRDXMS.SYS
				SRDXMSS.SYS
				XMSSIZE.EXE
				RAM Drive's driver's files

				UNISOUND.COM
				Legacy PnP sound card driver

				XCDROM.SYS
				CD-ROM driver
